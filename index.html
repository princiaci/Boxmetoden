Hvordan kommentere i HTML

HTML-kommentarer vises ikke i nettleseren, men de kan hjelpe til med å dokumentere HTML-kildekoden.

Eksempel: < ! - Skriv kommentarene dine her - >

Hvordan kommentere i CSS

Kommentarer brukes til å forklare koden, og kan hjelpe når du redigerer kildekoden på et senere tidspunkt.

Eksempel: / * Dette er en kommentar * /

Hvordan legger man til bilder?

For å legge til bilder bruker man koden

< img src=“” alt=“” >

Deretter skriver man ./filnavn i mellom “” “Alt” står for alternative text, som brukes til å forklare hva bildet er dersom det skjer noe feil.

Hvordan legger man til bakgrunnsfarge

Man endrer bakgrunnsfarge i CSS ved å spesifisere objektet med “objekt” {background-color: yellow;}

Hva brukes main til i HTML

Main brukes til å legge elementer som er unike for sidene, områder som ikke er unike (som sidebars) bør holdes utenfor, men innenfor “body”

Hvordan skriver man farge med HEX verdi

En heksadesimal farge er spesifisert med: #RRGGBB. RR (rød), GG (grønn) og BB (blå) er heksadesimale heltall mellom 00 og FF som spesifiserer intensiteten til fargen.

Hvordan skriver man farge med RGB verdi

En RGB-fargeverdi er spesifisert med: rgb (rød, grønn, blå). Hver parameter (rød, grønn og blå) definerer intensiteten til fargen som et helt tall mellom 0 og 255.

Hva brukes flexbox i CSS til?

Flexible Box Layout Module, gjør det enklere å designe fleksibel responsiv layoutstruktur uten å bruke flyt eller posisjonering.
